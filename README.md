# QR Code Generator
This python script will accept user's link and filename to create QR Code for themselves.
### How to use it?
1. Use your terminal, it could be powershell, cmd prompt, unix shell, or WSL.
2. Simply clone or download this repo.
3. Make python environment using ```python -m venv .venv``` or ```python3 -m venv .venv```
4. Activate python environment using ```source .venv/bin/activate``` for unix shell or ```.venv\Scripts\activate```
5. Install ```requirements.txt``` using ```pip install -r requirements.txt```.
6. Run ```python3 main.py``` or ```python main.py```
7. Enter your link.
8. Enter your file name.
9. It will create QR Code inside your current directory (your cloned repo directory).
10. You can see listed file in the console to track which file that you need.
11. QR Code is ready to use.
