# QR Code Generator
This python script will accept user's link and filename to create QR Code for themselves.
### How to use it?
1. Use your terminal, it could be powershell, cmd prompt, unix shell, or WSL.
2. Simply clone or download this repo.
3. Install ```requirements.txt``` using ```pip install requirements.txt```.
4. Run ```python3 main.py```
5. Enter your link.
6. Enter your file name.
7. It will create QR Code inside your current directory (your cloned repo directory).
8. You can see listed file in the console to track which file that you need.
9. QR Code is ready to use.
